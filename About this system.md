# 1. 背景

近年、糖尿病患者数は世界的に増加しており、早期発見および予防の重要性が高まっている。

本研究では、健康診断データに基づいて糖尿病リスクを予測する機械学習モデルを構築し、その有効性と説明可能性について検討した。

---

# 2. データセット

CDC Diabetes Health Indicators Dataset を使用した。

- サンプル数：253,680件
- 特徴量数：21

目的変数は Diabetes_binary であり、

- 0：糖尿病なし
- 1：糖尿病あり

とした。

---

# 3. 使用手法

本研究ではロジスティック回帰（Logistic Regression）を採用した。

ロジスティック回帰は各特徴量の影響度を係数として解釈できるため、医療分野で重要な説明可能性を有する。

---

# 4. 実験結果

| 指標 | 値 |
|--------|--------|
| Accuracy | 73.15% |
| Precision | 31.07% |
| Recall | 76.11% |
| F1-score | 44.13% |

Recall が高く、多くの糖尿病患者を検出できることが確認された。

一方で Precision は低く、偽陽性が比較的多い結果となった。

---

ロジスティック回帰モデルの係数を分析した結果を表1に示す。

これらの結果は既存の医学研究とも概ね一致している。

---

# 6. 考察

本研究により、BMI、年齢、高血圧、高コレステロールおよび総合健康状態が糖尿病リスクに強く関連していることが確認された。

また、ロジスティック回帰モデルを用いることで、各特徴量の影響度を係数として定量的に評価することができた。

これは単に糖尿病の有無を予測するだけでなく、どの要因がリスク上昇に寄与しているのかを明確に示せることを意味する。

医療現場では、AI が高リスクと判定した場合でも、その理由を説明できなければ医師や患者が結果を十分に信頼することは難しい。

本研究のモデルでは、「BMI が高い」「高血圧である」「年齢が高い」といった具体的な要因を提示できるため、医師が患者に対して生活習慣改善の必要性やリスク要因を説明する際の補助情報として活用できる可能性がある。

このように、本研究は予測性能だけでなく、医療AIにおいて重要な説明可能性（Explainable AI）を提供できる点にも意義があると考えられる。

---

# 7. 課題

- データ前処理手法のさらなる改善が必要である
- 特徴量選択の最適化について検討の余地がある
- モデル性能向上のためのハイパーパラメータ調整が十分ではない

---

# 8. 今後の展望

今後は、より多様な健康指標を取り入れた分析を行うことで、予測性能および実用性の向上を目指す。

また、医療現場における活用可能性について検討し、疾病予防や健康管理を支援するシステムへの発展を目指す。

---

# 9. 結論

健康診断データを用いた糖尿病リスク予測モデルを構築した結果、一定の予測性能を確認できた。

また、医療従事者へのヒアリングを通じて、予測結果だけでなく、その根拠を説明できることの重要性を認識した。

本研究では、BMI、年齢、高血圧などの特徴量が糖尿病リスクに与える影響を定量的に示すことで、医療現場における患者への説明を支援できる可能性を示した点に意義がある。

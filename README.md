# Diabetes Risk Prediction System

## Overview

This project predicts diabetes risk using machine learning models based on CDC health survey data.

## Dataset

CDC Diabetes Health Indicators Dataset

- Samples: 253,680
- Features: 21

## Model

- Logistic Regression

## Results

Accuracy: 74%

Recall: 73%

ROC-AUC: 0.81

## Feature Importance

- BMI
- Age
- High Blood Pressure
- High Cholesterol
- General Health

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
